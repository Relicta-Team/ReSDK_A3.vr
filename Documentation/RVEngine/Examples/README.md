# Примеры использования RVEngine

Практические примеры создания плагинов для RVEngine.

## Список примеров

1. **[Hello World](HELLO_WORLD.md)** - Простейший плагин
2. **[Custom Command](CUSTOM_COMMAND.md)** - Регистрация своей SQF команды
3. **[Event Handling](EVENT_HANDLING.md)** - Обработка игровых событий
4. **[Custom Type](CUSTOM_TYPE.md)** - Создание пользовательского типа
5. **[Advanced Plugin](ADVANCED_PLUGIN.md)** - Продвинутый плагин с интерфейсами

## Требования

- Visual Studio 2022
- CMake 3.8+
- RVEngine (собранный)
- Arma 3

## Структура примера

Каждый пример содержит:
- Полный исходный код
- CMakeLists.txt для сборки
- Объяснение каждого шага
- SQF код для тестирования
